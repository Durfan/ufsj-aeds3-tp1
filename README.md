Primeiro Trabalho Prático de AED's III.

![](./input.svg)

## Pre-requisitos

* GNU Compiler Collection [https://gcc.gnu.org/](https://gcc.gnu.org/)
* Gnuplot [http://www.gnuplot.info/](http://www.gnuplot.info/)

## Instalando/Removendo

Instalando:

```cmd
make
```

Removendo:

```cmd
make clean
```

## Executando

```cmd
./hcamp -i input -o output
```
